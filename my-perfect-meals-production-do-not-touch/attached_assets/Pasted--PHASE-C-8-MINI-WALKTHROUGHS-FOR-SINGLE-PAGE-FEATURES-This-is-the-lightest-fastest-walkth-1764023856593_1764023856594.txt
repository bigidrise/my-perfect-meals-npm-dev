# üî∑ **B. REQUIRED EVENT EMITTERS**

Every micro-walkthrough requires the same 3 events:

### **‚Äúopened‚Äù**

When the page loads and mounts.

### **‚Äúinteracted‚Äù**

When the user performs the main action (search, generate, calculate, save, add item, etc.)

### **‚Äúcompleted‚Äù**

When results appear / list updates / recipe opens.

Examples:

**Fridge Rescue**

```ts
window.dispatchEvent(new Event("opened"));
window.dispatchEvent(new Event("interacted"));
window.dispatchEvent(new Event("completed"));
```

**Macro Calculator**
‚Äúcompleted‚Äù fires when results appear.

**Master Shopping List**
‚Äúcompleted‚Äù fires when user adds at least one item.
